🧮 Academic GEO / AEO Formulas & Evaluation Framework
AI Citation Rate (CR)
CR = Σ(Outputs citing Domain D) / Total Outputs Q
Measures the likelihood of the domain being cited across a test set of generative queries. Source: Princeton/Georgia Tech GEO Research.
Share of Citation (SoC)
SoC = Citations to Domain D / Total Citations in Output
Calculates the percentage of the citations within a single generated answer pointing to your page.
Factual Density Limit
Density = Clean Visible Text Length / Raw HTML Size > 15%
Perplexity Rerankers discard pages with code-heavy markup enclosing sparse prose.
Structural Variance Score
Variance = |SSR Nodes - CSR Nodes| = 0
Symmetry target. Deviations between server-side HTML and client-side hydrated DOM raise the GPU Compute Tax.
